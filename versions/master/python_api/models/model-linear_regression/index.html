A new instance of LinearRegressionModel

Linear Regression [R1] is used to model the relationship between a scalar dependent variable and one or more independent variables. The Linear Regression model is initialized, trained on columns of a frame and used to predict the value of the dependent variable given the independent observations of a frame.

footnotes

[R1]https://en.wikipedia.org/wiki/Linear_regression
[R2]https://spark.apache.org/docs/1.5.0/ml-linear-methods.html
