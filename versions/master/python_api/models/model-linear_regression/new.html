: Model

A new instance of LinearRegressionModel

Linear Regression [R16] is used to model the relationship between a scalar dependent variable and one or more independent variables. The Linear Regression model is initialized, trained on columns of a frame and used to predict the value of the dependent variable given the independent observations of a frame. This model runs the MLLib implementation of Linear Regression [R17] with the SGD [R18] optimizer.

footnotes

[R16]https://en.wikipedia.org/wiki/Linear_regression
[R17]https://spark.apache.org/docs/1.3.0/mllib-linear-methods.html#linear-least-squares-lasso-and-ridge-regression
[R18]https://en.wikipedia.org/wiki/Stochastic_gradient_descent
