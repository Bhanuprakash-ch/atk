DaalNaiveBayesModel __init__¶

__init__(self, name=None)¶

[BETA] Create a ‘new’ instance of a multinomial Naive Bayes model

Parameters:

name : unicode (default=None)

User supplied name.
Returns:

: Model

A new instance of DaalNaiveBayesModel

Naive Bayes [R17] is a probabilistic classifier with strong independence assumptions between features. It computes the conditional probability distribution of each feature given label, and then applies Bayes’ theorem to compute the conditional probability distribution of a label given an observation, and use it for prediction. The Naive Bayes model is initialized, trained on columns of a frame, tested against true labels of a frame and used to predict the value of the dependent variable given the independent observations of a frame and test the performance of the classification on test data. This model runs the Intel DAAL implementation of Naive Bayes [R18].

footnotes

[R17]https://en.wikipedia.org/wiki/Naive_Bayes_classifier
