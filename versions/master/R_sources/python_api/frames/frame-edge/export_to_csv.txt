
:doc:`EdgeFrame <index>`  export_to_csv
***************************************

------


.. function:: export_to_csv(self, folder_name, separator=',', count=-1, offset=0)

    Write current frame to HDFS in csv format.


    :Parameters:

        **folder_name** : unicode

        ..

            The HDFS folder path where the files
            will be created.



        **separator** : unicode (default=,)

        ..

            The separator for separating the values.
            Default is comma (,).



        **count** : int32 (default=-1)

        ..

            The number of records you want.
            Default, or a non-positive value, is the whole frame.



        **offset** : int32 (default=0)

        ..

            The number of rows to skip before exporting to the file.
            Default is zero (0).




    :Returns:

        : dict

        ..

            The URI of the created file
    Export the frame to a file in csv format as a Hadoop file.

    Examples
    --------

    .. code::

        >>> frame.export_to_csv('covarianceresults')
        [===Job Progress===]
        "hdfs://hostname/user/user1/covarianceresults"
